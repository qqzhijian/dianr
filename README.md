# 点燃 (DianR) - 轻量级在线交友平台

**点燃生活，遇见未来**

点燃是一个专为用户、媒人和商家设计的轻量级社交交友平台，专注于核心交互功能，适配宝塔面板（PHP 8.2 + MySQL），服务器空间有限，采用最优轻量方案实现。

## 平台概述

### 核心理念
- **轻量高效**: 避免冗余功能，专注核心交互
- **隐私保护**: 手机号隐藏显示，资料查看需同意
- **三方互动**: 用户、媒人、商家角色差异化，紧密关联
- **安全可信**: 实名认证体系，评价系统保障质量

### 技术栈
- **后端**: 原生PHP 8.2 + MySQL
- **前端**: HTML + CSS + JavaScript + Bootstrap 5
- **架构**: MVC轻量架构，无重型框架

## 用户角色说明

### 1. 用户 (User)
普通交友群体，平台主要使用者。
- **注册**: 手机号/邮箱 + 昵称 + 密码
- **权限**: 查看活动、报名活动、在线聊天、申请查看他人详细资料
- **资料**: 性别/年龄/地区/简介（可选）

### 2. 媒人 (Mediator)
撮合角色，提供专业服务。
- **注册**: 同用户，可选实名认证
- **权限**: 发起活动、与用户/商家聊天、关联认证
- **资料**: 从业简介/擅长领域

### 3. 商家 (Merchant)
场景提供方，提供活动场地或服务。
- **注册**: 同用户，补充商家信息
- **权限**: 发起活动、与用户/媒人聊天、关联认证
- **资料**: 经营范围/地址

### 4. 管理员 (Admin)
后台专属角色，无前台注册。
- **权限**: 全平台管理、用户审核、活动管理、黑名单管理

## 核心功能

### 1. 注册登录
- **注册**: 简化的注册流程，仅需手机号/邮箱 + 昵称 + 密码，新用户默认普通用户角色
- **登录**: 手机号/邮箱 + 密码，支持验证码快捷登录
- **验证**: 新用户注册无需验证码，使用必要功能时提醒验证
- **界面**: 现代化的卡片式设计，渐变背景，响应式布局

### 2. 资料管理
- **隐私保护**: 前台仅显示昵称 + 隐藏中间5位的手机号（如138****1234）
- **详细资料**: 性别、年龄、地区、简介等，查看需对方同意
- **角色资料**: 媒人显示从业信息，商家显示经营信息
- **在线状态**: 实时显示在线/离线/离开状态

### 3. 活动功能
- **发起**: 仅媒人和商家可创建活动（标题/时间/地点/关联角色）
- **查看**: 用户可浏览所有活动
- **报名**: 用户可报名参加活动
- **管理**: 管理员可审核和管理活动

### 4. 在线聊天
- **类型**: 轻量级一对一文字聊天
- **记录**: 保存最近30天消息
- **状态**: 显示对方在线状态
- **限制**: 无群聊/文件传输，支持黑名单屏蔽

### 5. 互动认证
- **资料请求**: 用户可申请查看他人详细资料
- **关联认证**: 三方角色间可建立认证关系
- **评价系统**: 评价服务质量，确保平台可信度

### 6. 安全管理
- **黑名单**: 用户可手动拉黑，屏蔽交互
- **小黑屋**: 管理员可限制违规用户权限
- **权限控制**: 基于角色差异化权限

### 7. 后台管理
- **用户管理**: 查看、编辑、禁用用户
- **活动管理**: 审核、删除违规活动
- **数据统计**: 用户数、活动数等简易统计
- **黑名单管理**: 添加/移除黑名单用户

## 使用指南

### 首次使用
1. **访问网站**: 打开浏览器访问平台首页
2. **注册账号**: 点击"立即加入"，填写基本信息，选择角色
3. **验证账号**: 使用必要功能时按提示验证
4. **完善资料**: 登录后在个人资料页面填写详细信息

### 基本操作
1. **浏览活动**: 首页查看最新活动，或进入活动页面浏览全部
2. **参与活动**: 点击活动详情，点击"报名"参加
3. **查看用户**: 进入用户页面浏览其他用户
4. **发起聊天**: 在用户资料页或聊天页面选择联系人开始对话
5. **管理资料**: 在个人资料页面编辑和更新信息

### 高级功能
1. **创建活动**: 媒人/商家登录后，在活动页面创建新活动
2. **申请资料**: 在他人资料页申请查看详细资料
3. **拉黑用户**: 在聊天或用户页面添加黑名单
4. **评价服务**: 在评价页面对服务进行评价

## 部署说明

### 1. 环境准备
```bash
# 安装PHP 8.2和MySQL
# Ubuntu/Debian
sudo apt update
sudo apt install php8.2 php8.2-mysql mysql-server nginx

# CentOS/RHEL
sudo yum install php82 php82-mysql mysql-server nginx
```

### 2. 数据库设置
```sql
-- 创建数据库
CREATE DATABASE dianr CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 导入表结构
mysql -u root -p dianr < database.sql
```

### 3. 代码部署
```bash
# 克隆或上传代码到网站根目录
# 例如 /var/www/html/dianr

# 设置权限
chown -R www-data:www-data /var/www/html/dianr
chmod -R 755 /var/www/html/dianr
```

### 4. 配置Nginx
```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /var/www/html/dianr;
    index index.php;

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location ~ \.php$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/var/run/php/php8.2-fpm.sock;
    }

    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}
```

### 5. 配置宝塔面板
1. 在宝塔面板中添加网站
2. 设置域名和根目录
3. 安装PHP 8.2和MySQL
4. 导入数据库
5. 配置SSL证书（推荐）

### 6. 初始化设置
1. 编辑 `config/config.php`，更新数据库连接信息

## 常见问题

### Q: 注册时提示手机号已存在怎么办？
A: 检查是否已注册过账号，尝试找回密码或联系管理员。

### Q: 无法发送消息怎么办？
A: 检查账号是否已验证，以及对方是否在黑名单中。

### Q: 活动报名后如何取消？
A: 当前版本不支持取消报名，请谨慎报名。

### Q: 忘记密码怎么办？
A: 联系管理员重置密码，或重新注册新账号。
