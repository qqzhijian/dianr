# 点燃 - 点燃生活，遇见未来

一个基于PHP 8.2 + MySQL的轻量级在线交友平台，适配宝塔面板，支持用户、媒人、商家三方角色交互。

## 功能特性

### 核心角色
- **用户**：普通交友群体，支持极简注册和基本资料展示
- **媒人**：撮合角色，可发起活动和提供专业服务
- **商家**：场景提供方，可发起活动和提供场地服务
- **管理员**：后台管理系统管理员

### 主要功能
- 用户注册登录（手机号+验证码）
- 邮箱/手机号验证系统（支持无验证注册，提升转化率）
- 资料展示与隐私保护（需同意查看详细资料）
- 活动发布与参与（媒人商家需验证后发布）
- **聊天大厅与私聊功能**（实时在线聊天，支持消息通知）
- 媒人商家认证体系
- 用户评价系统
- 黑名单与小黑屋管理
- 后台数据统计与管理

## 技术栈

- **后端**：PHP 8.2 原生开发
- **数据库**：MySQL 5.7+
- **前端**：HTML + CSS + JavaScript + Bootstrap 5
- **服务器**：适配宝塔面板

## 验证系统

平台采用渐进式验证策略，提升用户转化率：

### 注册流程
- 支持邮箱或手机号注册，无需立即验证
- 注册后可正常浏览平台内容
- 享受基础功能（查看公开资料、浏览活动等）

### 验证要求
以下功能需要验证邮箱/手机号后才能使用：
- **聊天功能**：发送消息、查看聊天记录
- **活动发布**：媒人/商家创建活动
- **好友申请**：申请查看他人详细资料

### 验证流程
1. 用户注册后访问需验证功能时，系统自动跳转到验证页面
2. 输入6位数字验证码（演示环境接受任意6位数字）
3. 验证成功后解锁相应功能
4. 验证状态永久保存

## 聊天系统

平台提供完整的聊天解决方案，支持实时沟通：

### 聊天大厅
- **在线用户列表**：显示最近10分钟活跃用户
- **实时状态更新**：每30秒刷新在线状态
- **快速发起聊天**：点击用户直接开始对话
- **未读消息提醒**：显示未读消息数量

### 私聊功能
- **消息历史**：保存30天聊天记录
- **实时消息推送**：3秒轮询新消息
- **消息状态**：已读/未读状态管理
- **消息通知**：新消息桌面通知

### 聊天入口
- 导航栏：聊天大厅入口
- 首页：登录用户快速入口
- 个人资料页：私聊按钮
- 活动页面：与组织者聊天

- 用户状态验证
- 防止恶意消息轰炸
